## Program Execution:

1. Compile a java program using:
`javac filename.java`

2. Execute a java program using:
`java filename`

## Dependencies:

### 1/2 Java Runtime Environment 

- Install for Ubuntu/Debian
``` 
    sudo apt-get update 
    sudo apt install default-jre
```

### 2/2 Java Development Kit

- Install for Ubuntu/Debian
```
    sudo apt install default-jdk
```
